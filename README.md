# Re-Mmogo Web App

Project Description

The frontend application for [Re-Mmogo Web App] -The comprehensive web plartform designed to help motshelo groups manage their finances effectively. The React-base interface provides an intuitive dashboard for group administators and members to track contributions, manage loans and generate financial records.

Problem Solved

Traditional motshelo groups strugle with manual record keeping, leading to disputes and inefficiency.Re-Mmogo digitizes the entire savings group management process ensuring transparency, accuracy, and ease access to all enrolled members.


Main Features:

Group Registration: Create and register new Motshelo groups
Member Management: Enroll and manage group members
Contribution Tracking: Record monthly contributions (P1000 per member)
Loan Management: Apply for and track loans with 20% monthly interest calculation
Approval Workflow: Dual signatory approval system for loans and payments
Payment Processing: Member-initiated payments with signatory verification
Financial Reporting: Year-end reports showing individual member payouts, interest generation (minimum P5000 target per member).
Responsive Dashboard: Real-time balance tracking for each member.
